CyberSecurity Web App

"I spearheaded the UX redesign of a cybersecurity path analysis tool, focusing on turning complex technical data into an intuitive learning experience. My objective was to solve for accessibility without compromising depth. I achieved this by developing a visual discovery-based UI that replaces traditional, high-friction forms with an interactive network map. Key achievements include the creation of contextual detail panels that 'teach' the user industrial terminology (like PLCs and HMIs) and a guided path-setup flow that assists interns in identifying destinations they previously wouldn't have known how to find.
